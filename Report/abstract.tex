\newenvironment{Abstract}
\vspace{20pt}
This project implemented an intelligent vehicle surveillance system that records and uploads video only when a relevant event occurs. The system was designed to improve vehicular security and situational awareness by combining event detection, buffered video capture, cloud upload, and mobile access within a modular architecture. Instead of relying on continuous recording, the system preserves a rolling pre-event buffer and stores post-event footage so that each incident is captured with adequate context while reducing unnecessary storage usage.\par
The prototype was developed on a Linux-based platform using independent modules for event detection, event coordination, video recording, cloud upload, and mobile access. Trigger events such as honking, sudden braking, alarm activation, and suspicious person detection initiate recording. Video is encoded into MP4 format, uploaded to a server, and made available through an Android application that also receives event notifications. The implementation demonstrates the feasibility of integrating computer vision, multimedia processing, cloud services, and mobile connectivity into a practical surveillance workflow.\par
The completed system provides event-driven recording, remote access to incident footage, and real-time user notification. The report discusses the design decisions, implementation details, functional validation, and limitations of the prototype, and shows how the proposed approach offers a more efficient and responsive alternative to passive dash-camera systems.
